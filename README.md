# Calculator

A responsive calculator built with HTML, CSS, and vanilla JavaScript.  
Supports basic arithmetic operations with error handling and animated UI feedback.

## Features
- Addition, subtraction, multiplication, division
- Decimal support
- Division-by-zero error handling (Infinity/Error states)
- Responsive design
- Animated “broken” effect on invalid operations

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES5)

## Live Demo
https://nguyenvan22.github.io/javascript-calculator/

## Run Locally
1. Download or clone the repo
2. Open `index.html` in a browser
